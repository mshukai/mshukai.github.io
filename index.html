Obsidian 使用教程

与其他 Markdown 软件相比，Obsidian 有以下优势：

  1. 软件开源，可以免费使用；
  2. 功能强大，插件机制且拥有强大的第三方插件市场；
  3. UI 界面好看，软件使用过程中流畅无卡顿。

如果你倾向于使用 Typora，可以参考文章：[[玩转Typora]]。

内置功能探索

  • 如何在 Obsidian 中优雅的进行注释（边注） - 知乎 (zhihu.com)

更多功能可参考 Obsidian 中文帮助。

软件安装与插件市场

电脑上安装软件就不用多说了，下面给出官方下载地址：Obsidian 下载链接。

如何访问社区插件市场

Obsidian 在国内是无法访问第三方插件市场的，需要借助工具才能下载社区插件进行安装，其中有一个非常流行的 CDN 加速插件 PKMer，是专门用于解决插件市场无法访问的问题，这里是插件的主页。关于如何安装和使用 PKMer 插件，可以参考这篇[文章](PKMer 插件电脑端安装手把手教程)。

[!NOTE] 访问插件市场的其它方案
也不一定非要使用 PKMer 插件实现对第三方插件市场的访问，比如使用其它的 Github 加速软件如 fastgithub 等。具体可以参考这篇文章。

好用的插件推荐

Editing Toolbar

一款 Markdown 可视化编辑工具，主要有以下特点：

  • 支持命令面板自定义新增、删除项，自定义命令分组、拖拽调整顺序等；
  • 支持格式刷功能，包括字体颜色和背景颜色；

[!NOTE] 提示
如果想要换一个 UI 风格，还可以在主题市场选择一个主题进行安装（PKMer 插件提供的功能包括插件市场和主题市场）。

使用快捷键

Obsidian 支持快捷键自定义设置，但是默认只设置了没几项功能，下面通过表格的形式列出所有常用的功能快捷键（最基本的除外，比如：查找、保存等操作）。

格式相关

格式 快捷键 是否自定义
标题级别切换/取消标题 Ctrl+数字 是（需借助 Editing Toolbar 插件）
取消标题 Ctrl+反引号
字体加粗 Ctrl+B
字体下划线 Ctrl+U 是（需借助 Editing Toolbar 插件）

Markdown 书写相关

功能 快捷键 是否需要自定义（备注）
插入链接 Ctrl+K
插入内部链接 [[
插入标注 Ctrl+Shift+B
插入行内代码 一对反引号
插入代码块 Ctrl+Shif+K
跳转到文章开头 Ctrl+Home
跳转到文章结尾 Strl+End
插入行内数学公式 Ctrl+M 是
插入数学公式块 Ctrl+Shift+M 是（注意：和搜狗输入法冲突）
删除当前行 Alt+Delete
删除当前段落 Ctrl+D

表格相关（全部需要设置）

操作 快捷键
/
 操作 快捷键
插入表格 Ctrl+T 下方新增行 Ctrl+Enter（有冲突）
删除行 Ctrl+Shift+Delete 上方插入行 Ctrl+Shift+Enter
上下移动行/列 Alt+方向键 复制当前行 Ctrl+Shift+C
右侧新增列 Alt+Enter（有冲突） 删除列 Alt+Shift+Delete
左边插入列 Alt+Shift+Enter 复制当前列 Alt+Shift+C
列左/右对齐 Ctrl+Shift+方向键 列居中对齐 Ctrl+Shift+Space

其它功能

功能 快捷键
新标签打开文件/内链 Ctrl+鼠标左键
转到下一个标签 Ctrl+Tab
转到上一个标签 Ctrl+Shift+Tab

[!NOTE] 提示
简单的 Markdown 原生语法就没必要使用快捷键了，比如：分割线、引用、无序/有序列表等，而且除了使用 Markdown 原生语法和快捷键，别忘了还有强大的可视化编辑插件。有些功能貌似没有快捷键但是写法稍显复杂，如任务列表的语法 - [x]，这时候直接使用可视化编辑插件插入就行了。
