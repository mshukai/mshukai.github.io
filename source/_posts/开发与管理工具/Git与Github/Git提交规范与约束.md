## Git 提交语法

> git commit <type>(scope): subject
> description: message detail

## type 取值规范

| 取值     | 意义描述                 |
| -------- | ------------------------ |
| feat     | 增加新功能               |
| fix      | 修复 bug                 |
| style    | 修改代码风格，不影响功能 |
| refactor | 重构代码                 |
| test     | 测试代码                 |
| perf     | 性能提升、体验优化等     |
| chore    | 构建                     |
| revert   | 功能回滚                 |
| docs     | 文档维护                 |
